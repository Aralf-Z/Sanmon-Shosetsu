## About the Name

This is my personal C# game framework for Unity.  
The name **Sanmon** comes from the romanized Japanese title of the song *三文小説* by King Gnu.

As the title implies, this framework is my own “three-penny novel.”  
Just like life itself, it may be imperfect, rough, or inexpensive in value, but it is still a story that belongs to me.
